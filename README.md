# Proiect-Abstract-Factory
Proiect PIP semestrul 1

The main selector menu. There are 2 menus from each a option can be selected. Once selected a material and a tool type, the user can click a button to craft the wanted item.
Once the "Craft Item" button is triggered, an image resembling the tool with the configuration selected will appear (see pictures below).
